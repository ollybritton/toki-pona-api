Toki Pona API

This is an open-source JSON API for looking up information associated with the oligoisolating constructed language Toki Pona, a language designed to be as simple and as easy to learn as possible.

Endpoint #1: /word/<string:word>
Look up information associated to a word.

This endpoint allows you to find information associated with the word supplied. It returns an JSON list containing a set of objects, each object representing a word that could possibly be linked to the word given.

Example requests
/word/a
 
/word/ijo
 
/word/pakala
 
/word/xyz
Endpoint #2: /data
Get the raw data of the API.

The endpoint allows you to get the raw data from the API, a json object of word objects.

Example requests
/data
Endpoint #3: /random
Get a random word.

This endpoint allows you to fetch a random word from the API.

Example requests
/random
